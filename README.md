# 崩坏：星穹铁道来古士删除数据模拟器

这是一个模拟《崩坏：星穹铁道》中来古士删除数据过程的Python程序。该程序通过图形界面模拟了数据删除时的视觉效果，包括错误信息滚动显示、进度条以及文件名显示等元素。

## 功能说明

- **全屏显示**：程序启动后会以全屏模式运行，覆盖整个屏幕。
- **错误信息滚动**：在屏幕左侧显示模拟的错误信息，这些信息会不断滚动，模拟数据删除过程中出现的错误提示。
- **进度条显示**：在屏幕下方显示一个进度条，模拟数据删除的进度。
- **文件名显示**：在进度条上方随机显示文件名，模拟正在删除的文件。
- **ESC退出**：按ESC键可以退出程序。

## 文件结构说明

- `Pillow的方法.py`：主程序文件（版本1），使用Pillow库进行文本渲染，包含所有模拟逻辑和界面绘制代码。
- `test.py`：主程序文件（版本2），使用tkinter内置字体进行文本渲染，功能与`Pillow的方法.py`相同。
- `error.txt`：包含模拟错误信息的文本文件。
- `name.txt`：包含模拟文件名的文本文件。
- `requirements.txt`：Python依赖库列表。

## 安装和使用说明

### 安装依赖

在运行程序之前，需要安装所需的Python库。可以通过以下命令安装：

```bash
pip install -r requirements.txt
```

### 运行程序

运行程序非常简单，只需执行以下命令：

```bash
python "Pillow的方法.py"
```

或

```bash
python test.py
```

程序启动后会进入全屏模式，模拟数据删除过程。按ESC键可以退出程序。

## 文件详细说明

### Pillow的方法.py

这是程序的主文件（版本1），使用了Python的tkinter库来创建图形界面，并使用Pillow库来处理文字渲染。程序主要包含以下几个部分：

1. **FixedTransparentErrorSimulator类**：这是程序的主类，负责创建全屏窗口、绘制界面元素、处理文本滚动和进度条更新等。
2. **字体加载**：程序会尝试加载系统字体，如果找不到合适的字体则使用默认字体。
3. **文件读取**：程序会读取`error.txt`和`name.txt`文件中的内容，用于显示错误信息和文件名。
4. **界面绘制**：包括边框、进度条、警告图标等元素的绘制。
5. **文本渲染**：使用Pillow库将文本渲染为图片，然后在Canvas上显示。
6. **文本滚动**：实现错误信息的滚动显示效果。
7. **进度更新**：模拟进度条的更新过程。
8. **文件名显示**：随机显示文件名，模拟删除过程。

### test.py

这是程序的主文件（版本2），与`Pillow的方法.py`功能完全相同，但使用tkinter内置的字体渲染功能，而不是Pillow库。主要区别包括：

1. **文本渲染方式**：使用tkinter内置的`create_text`方法，而不是Pillow渲染为图片。
2. **依赖库**：虽然此版本不直接使用Pillow，但仍需安装以便与`Pillow的方法.py`保持兼容。
3. **功能**：实现完全相同的效果，包括错误信息滚动、进度条更新和文件名显示。

### error.txt

该文件包含模拟的错误信息，每行一条错误信息。程序会循环读取这些信息并显示在屏幕上。

### name.txt

该文件包含模拟的文件名，每行一个文件名。程序会随机选择这些文件名并显示在进度条上方。

### requirements.txt

该文件列出了程序运行所需的Python库：

```
Pillow
```

## 使用注意事项

1. 程序会以全屏模式运行，覆盖整个屏幕，请确保在运行前保存好其他工作。
2. 按ESC键可以退出程序。
3. 两个版本的程序（`Pillow的方法.py`和`test.py`）功能相同，可根据个人喜好选择使用。
4. 程序依赖于Pillow库（即使使用`test.py`版本），请确保已正确安装。
5. 程序在Windows系统上测试通过，其他系统可能需要调整字体路径。

## 结语

这个模拟器是为了重现《崩坏：星穹铁道》中来古士删除数据时的视觉效果而创建的。它并不真正删除任何文件，仅用于演示和娱乐目的。提供了两个版本（`Pillow的方法.py`和`test.py`），可选择其中一个运行。
